# MSPM0 Custom PCB Design

This repository contains the complete design files for a **custom MSPM0-based embedded PCB** designed using **KiCad**.  
The project includes schematic design, PCB layout, routing files, and supporting documentation required to manufacture the board.

The PCB is designed with a compact layout and optimized component placement suitable for embedded and IoT applications.

---

## Project Overview

This project demonstrates the complete workflow of **custom PCB design**, starting from schematic capture to PCB layout and final fabrication files.

The design includes:
- MSPM0 microcontroller based system
- Power supply circuitry
- Passive components and decoupling capacitors
- Custom component placement
- Compact PCB layout
- Design rule checks (DRC/ERC)
- Routing and board optimization

The PCB layout and component placement can be seen in the project documentation. :contentReference[oaicite:0]{index=0}
